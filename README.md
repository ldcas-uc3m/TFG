## Installation and execution

Compile
- Linux:
    ```bash
    cd build/
    cmake ..
    make
    ```
- Windows (PowerShell)
    ```powershell
    cd build/
    cmake -G Ninja ..
    cmake --build .
    ```

Run
```bash
./src/tfg
```
